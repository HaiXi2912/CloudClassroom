# 02 本地初始化与验证（MySQL 8）

> 目标：把数据库与表结构在你本机 MySQL 跑通。

## 1. 连接信息（本项目默认）

- 主机：127.0.0.1
- 端口：3306
- 用户：root
- 密码：123456

## 2. 一键初始化（PowerShell）

在项目根目录执行：

```powershell
cd /d d:\CloudClassroom

# 建库
Get-Content -Raw .\sql\001_create_database.sql | mysql -h 127.0.0.1 -P 3306 -uroot -p123456

# 建表
Get-Content -Raw .\sql\002_create_tables.sql | mysql -h 127.0.0.1 -P 3306 -uroot -p123456

# 初始化数据
Get-Content -Raw .\sql\003_seed_data.sql | mysql -h 127.0.0.1 -P 3306 -uroot -p123456
```

## 3. 验证 SQL

```sql
USE cloud_classroom;
SHOW TABLES;
SELECT role_code, role_name FROM sys_role;
SELECT username, nickname FROM sys_user;
```

## 4. 常见问题

- 连接失败：请确认 MySQL 服务已启动、root 密码是否为 123456。
- 权限不足：请使用有建库/建表权限的账号执行（课堂环境通常用 root）。
